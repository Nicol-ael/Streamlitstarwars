## 📖 Descripción

Dashboard interactivo desarrollado con Streamlit para analizar personajes del universo Star Wars.

Permite explorar datos mediante filtros dinámicos y visualizaciones interactivas, facilitando el análisis de patrones como facción, altura, popularidad y sensibilidad a la Fuerza.

---

## ✨ Características

* ✅ Filtros dinámicos
* ✅ KPIs (métricas)
* ✅ Gráficos interactivos con Plotly
* ✅ Búsqueda por nombre
* ✅ Diseño personalizado estilo Star Wars

---

## 🛠️ Tecnologías

* Python
* Streamlit
* Pandas
* Plotly

---

## ⚙️ Instalación

```bash
git clone https://github.com/Nicol-ael/starwars-dashboard.git
cd starwars-dashboard
pip install -r requirements.txt
streamlit run app.py
```

---

## 💡 Uso

1. Ejecuta la app
2. Usa los filtros del panel lateral
3. Explora gráficos y métricas
4. Analiza los datos
